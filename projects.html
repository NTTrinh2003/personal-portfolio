Personal Portfolio

Project description:

  • This project is to show what can I do. The past works, the experience, the skills, they are all in here.

  • For the employer, this project is to validate my value on the proposed CV.

  • The functions are simple. On the main page, click on the wanted content you want to view to redirect to the content. If you want to view my skills, projects or contact info, click on the tab on the top right.

In this project I have used:

  • Plain HTML, CSS, JavaScript.

  • ChatGPT and Google for styles suggestions.

  • Zencoder for repeatable code.

  • Github for version control.

What I learned during the project:

  • Semantic HTML, SEO Meta tags.

  • CSS Grid Layout.

  • Responsive Web Design.

The project template is from roadmap.sh frontend project
